Snow CLI Commands¶

Usage¶

To connect to your ServiceNow instance the environment variables SNOW_USERNAME, SNOW_INSTANCE, SNOW_PASSWORD are used. The authentication used is JSONv2.

snowcli¶

snowcli [OPTIONS] TABLE ACTION

Options

--param <param>¶

Used to pass paramaters to the servicenow query.

--custom-table¶

Used to interact with data in custom table

Arguments

TABLE¶

Required argument

ACTION¶

Required argument
